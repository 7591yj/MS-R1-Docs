# MS-R1 — 调试命令

MS-R1 设备故障排查时常用调试命令的简明参考。

## 目录
- [GPU](#gpu)
- [VPU（视频处理单元）](#vpu视频处理单元)
- [注意事项](#注意事项)

## GPU

此命令显示 GPU 利用率（时钟 / 跟踪 信息）：

已知问题：当 GPU 未被使用时，百分比数字会保持不变（冻结）。

```bash
gpu_utilization_clock_tracing
```

## VPU（视频处理单元）

此命令显示 VPU的试用情况  
如果视频编码/解码使用 VPU，会显示大量日志：

```bash
mvx_logd --follow
```

## 注意事项

- 确保你具有运行这些命令所需的权限（root）。
- 仅将这些命令用于调试目的。
